# 加热方式

对于喷出的热水的：

* 常见加热方式
  * **储热式**：最老的，带水箱，热得快的原理一样去加热，加热慢
  * **速热式**：普通的，带小水箱(常见容量`0.4L`)，也要加热一小会后，才有热水
  * **即热式**：最好的，不带水箱，立即加热出热水。一般用**陶瓷加热芯片**实现实时加热
* 对比=结论
  * <font color=red>即热式 >> 速热式 > 储热式</font>
