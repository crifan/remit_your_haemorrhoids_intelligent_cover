# 座圈抗菌

为了（尤其是多人共用时）座圈更加卫生和安全，有些加了抗菌科技，使得座圈沾染细菌的概率大大降低。

* 常见抗菌科技的效果对比是：
  * <font color=red>银离子抗菌 < SIAA抗菌 < 纳诺怡抗菌</font>
